# Career-OS

This repository contains the code and documentation for the **Career OS** project, a personal career assistant web application built using Next.js, Node.js, and Google Cloud services.

## Overview

Career OS helps a single user manage job searches, networking, scheduling, and application pipelines in one place. It integrates with Gmail, Google Calendar, and other job sources to classify emails, track job postings, create follow-up tasks, and provide résumé and interview preparation tools.
