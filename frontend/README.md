# Frontend

This directory contains the frontend of the Career OS application, built with Next.js and TypeScript. It includes React components, pages, and hooks for the user interface.

## Structure

* `pages/` - Next.js pages for routing.
* `components/` - Reusable React components.
* `styles/` - CSS or Tailwind configuration.

## Development

Install dependencies and run the development server:

```
npm install
npm run dev
```

The app will be available at http://localhost:3000.
